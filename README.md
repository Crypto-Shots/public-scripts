# INSTRUCTIONS


#### Install dependencies

```
npm i
```


## HIVE

-----

#### Print Hive blocks
```
npm run scan-hive
```

#### Airdrop tokens
```
npm run airdrop-hive-tokens
```

#### Airdrop NFTs
```
npm run airdrop-hive-nfts
```

#### Lookup user's NFTs
```
npm run fetch-hive-nfts
```
